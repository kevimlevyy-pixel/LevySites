SITE CRIADOR DE SITES - TONY SITES

Arquivos:
- index.html
- style.css
- script.js
- privacidade.html

O que alterar antes de publicar:
1. No index.html, troque "Tony Sites" pelo nome da sua marca, se quiser.
2. No script.js, troque 5599999999999 pelo seu WhatsApp real.
3. No index.html, altere textos, serviços e exemplos de portfólio.
4. No privacidade.html, revise o texto da política conforme sua realidade.

Como publicar no GitHub Pages:
1. Crie um repositório no GitHub.
2. Envie estes arquivos para a raiz do repositório.
3. Vá em Settings > Pages.
4. Em Source, escolha Deploy from a branch.
5. Escolha main e /root.
6. Salve e aguarde o link ficar disponível.
