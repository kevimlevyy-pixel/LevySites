Política de Privacidade

Esta página explica de forma simples como as informações de contato são utilizadas neste site.

1. Dados coletados

O formulário pode solicitar nome, WhatsApp, tipo de site desejado e uma mensagem enviada pelo visitante.

2. Como os dados são usados

As informações são usadas apenas para responder ao pedido de orçamento e iniciar atendimento pelo WhatsApp.

3. Armazenamento

Este site estático não possui banco de dados próprio. Ao clicar no botão de envio, a mensagem é aberta no WhatsApp do visitante.

4. Compartilhamento

As informações não são vendidas. Elas podem ser compartilhadas apenas quando necessário para atendimento ou cumprimento de obrigação legal.

5. Contato

Para solicitar alteração ou remoção de informações enviadas, entre em contato pelo WhatsApp informado no site.
